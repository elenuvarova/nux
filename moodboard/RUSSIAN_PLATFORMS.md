# NUX — Исследование российских стриминговых платформ

Детальный разбор UI/UX каждой платформы: что работает, что не работает, что можно взять в NUX.

---

## 1. КИНОПОИСК (Яндекс)

**Позиция:** Лидер рынка, 14.1 млн пользователей, доля 32.7%
**Ссылка:** kinopoisk.ru

### Что делают хорошо

**Метаданные — лучшие на рынке**
Кинопоиск вырос из базы данных (аналог IMDb), поэтому метаданные — главное преимущество.
На странице фильма: рейтинг КП + IMDb, детальный актёрский состав, съёмочная группа, награды, бюджет, сборы, страна, студия, язык. Можно кликнуть на режиссёра → все его фильмы. На актёра → вся фильмография.

**Социальный слой — уникальная фича**
- "Смотрят ваши друзья" — подборка на основе активности друзей
- Возможность оценить, написать рецензию, обсудить в комментариях
- UGC: зарегистрированные пользователи могут редактировать базу данных
- Рейтинг начиная с 500 голосов (против 25.000 у IMDb) — быстрее накапливается для нишевых фильмов

**Редакционные материалы**
Новостной и медиа-раздел с рецензиями и статьями. В статьях — объектные карточки фильмов (inline-превью тайтла внутри текста). Связь редакционного контента с каталогом.

**Цветовое кодирование доступности**
Визуальный индикатор прямо на карточке: доступен ли фильм для онлайн-просмотра, требует ли подписку или покупку.

**Интеграция с Яндексом**
- Голосовой поиск ("Алиса, покажи фильмы Тарковского")
- Единый аккаунт Яндекс → меньше барьеров входа для текущих пользователей экосистемы
- Интеграция с Яндекс Музыкой (саундтреки к фильмам)

### Что не работает

**Дизайновая непоследовательность**
Новая главная страница резко контрастирует со старыми разделами. UGC-секции (рецензии, форумы) выглядят как интерфейс начала 2000-х. "Legacy code" никто не рефакторил — дизайн неоднородный по всему сайту.

**Закрытость без Яндекса**
Авторизация требует аккаунт Яндекс — барьер для пользователей без него.

**Перегруженность информацией**
Страница фильма очень насыщена — рейтинги, трейлеры, описание, актёры, отзывы, похожие — всё сразу. Информационная иерархия слабая.

### Что взять в NUX
- ✅ Богатые метаданные: режиссёр → filmography, актёр → другие фильмы на платформе
- ✅ Inline карточки фильма в редакционных текстах
- ✅ Визуальный статус доступности (доступен / в подписке / покупка)
- ✅ Социальный слой: "что смотрят похожие на тебя" (облегчённая версия)
- ❌ Не брать: дизайновую непоследовательность, перегруженность страницы

## 2. IVI

**Позиция:** 2-й по аудитории, ~10 млн, лучший поиск среди российских
**Ссылка:** ivi.ru

### Что делают хорошо

**Самый удобный поиск**
Пользователи стабильно отмечают поиск ivi как лучший среди российских платформ. Быстрый, работает с опечатками, поддерживает поиск по актёру/режиссёру.

**Кроссплатформенная дизайн-система (технически лучшая на RU-рынке)**

ivi построила впечатляющую систему:
- **JSON-формат дизайн-токенов** — одно описание компонента конвертируется во все платформы: iOS, Android, Web, Smart TV (Samsung/LG), tvOS, Apple TV, Xbox, PS4, Windows, Roku
- **Абстрактные имена для токенов**: цвета названы в честь городов (фирменный красный = "madrid", белый = "sofia", лиловый = "dublin"), типографика — в честь нимф. Это позволяет переименовать значение без переименования токена
- **729 иконок** в 122 цветовых вариациях
- **202 компонента** с состояниями (hover, pressed) и модификаторами
- Deprecation-трекер: 278 устаревших сущностей, автоматические уведомления в Slack разработчикам

**Видео-first карточки**
После редизайна карточки перешли с постеров к видео-превью. Тест показал прямую связь конверсии с видео-подходом — "гораздо активнее смотрят, когда карточка основана на видеоматериалах".

**Профиль — модульные тайлы**
Профиль пользователя построен на модульной системе плиток, адаптивных под разные размеры (телефон / сайт / ТВ).

**Регистрация через "чат"**
Нестандартное решение: многошаговые сценарии регистрации переведены в формат диалога/чата, что сокращает воспринимаемую сложность.

### Что не работает

**Cluttered интерфейс на бесплатном тарифе**
Реклама и смешение платного/бесплатного контента без чёткого разграничения — главная жалоба.

**Нет лёгкой темы**
Только тёмный интерфейс, как и у всех конкурентов.

### Что взять в NUX
- ✅ JSON дизайн-токены с абстрактными именами (прямо применимо)
- ✅ Видео-карточки с превью (уже в плане)
- ✅ Чат-интерфейс для онбординга (регистрация как диалог)
- ✅ Модульный профиль адаптивный под устройства
- ✅ Подход к поиску — быстрый, с поддержкой имён

## 3. OKKO (Сбер / Rambler)

**Позиция:** 3-й по доле (15.1%), лучший рост (+34%), лучший детский раздел
**Ссылка:** okko.ru

### Что делают хорошо

**"Воздушный" интерфейс — лучший среди российских**
Пользователи и обзорщики единодушны: Okko — самый лёгкий и незагромождённый интерфейс. Весь контент доступен в максимум **2 клика**. Навигация без лишних элементов, ничто не отвлекает от выбора.

**Интерактивное телевидение — киллер-фича**
Уникальная функция на рынке:
- Прямой эфир ТВ-каналов можно **поставить на паузу**
- **Перемотать** уже показанный эфир
- **Архив** — посмотреть передачу, которая шла 3 дня назад
- Перемотать рекламу (в записи)

**Мультикамерный спорт**
Во время спортивных трансляций (футбол, Олимпиада) пользователь сам переключает камеры, смотрит повторы, следит за статистикой в реальном времени. Нет аналогов у конкурентов.

**Детский раздел — лидер рынка**
Отдельный тематический интерфейс с лёгкой и светлой подачей, ориентированной на детей и родителей. Другие платформы не проработали детский раздел так же.

**Маскот как навигационный элемент**
Персонаж-маскот Okko участвует в навигации: даёт подсказки, рекомендует контент, меняет облик по сезону/тематике. Геймификация без излишества.

### Что не работает

**Слабый поиск**
В сравнении с ivi поиск Okko хуже — меньше фильтров, медленнее работает по именам.

**Проблемы с биллингом**
Жалобы на несанкционированные списания после отмены подписки. Проблема с UX отмены.

**Перегруженность в пиковые нагрузки**
Во время крупных спортивных событий сервис притормаживает.

**Тёмная тема обязательна**
Нет выбора, только тёмный интерфейс.

### Что взять в NUX
- ✅ "2-click" принцип — любой контент не дальше двух действий
- ✅ "Воздушный" подход — минимум хрома, максимум контента
- ✅ Маскот / редакционный персонаж (для NUX: голос куратора)
- ✅ Детский профиль — отдельный визуальный язык
- ❌ Не брать: UX отмены подписки (исправить как наоборот)

## 4. KION (МТС)

**Позиция:** ~10–12% рынка, приоритет на оригинальный контент
**Ссылка:** kion.ru

### Что делают хорошо

**Визуальная эстетика**
Из всех российских платформ KION имеет наиболее современный и "дорогой" визуальный дизайн. Команда МТС Design System вложила в эстетику больше, чем конкуренты.

**Детский профиль с PIN**
Отдельный детский профиль, защищённый PIN-кодом. Родители не должны переключаться между аккаунтами.

**Умные рекомендации**
Заявлена "умная система рекомендаций" — персонализация на базе истории просмотров.

**Оригинальный контент**
25 оригинальных проектов в 2025 году (конкурирует с Premier).

### Что не работает

**Критические проблемы производительности**
Это главная боль KION:
- Каталог "подвисает" при листании
- Страницы открываются **5 секунд** (недопустимо по любым UX-стандартам)
- Интерфейс красивый, но функционально медленный

Руководитель UX-команды KION сам признал: "лишние две секунды ожидания могут отбить желание взаимодействовать с платформой" — но проблема остаётся нерешённой.

**Style over function**
То же, что у HBO Max в западном контексте: ставка на визуальную красоту в ущерб скорости и удобству.

### Что взять в NUX
- ✅ Визуальная планка эстетики — KION доказывает, что российская аудитория ценит красивый дизайн
- ✅ Цитата руководителя KION как мантра разработки: "2 лишних секунды = потеря пользователя"
- ❌ Не брать: скорость как жертву красоте

## 5. PREMIER (ГПМ РТВ / ТНТ / ТВ-3)

**Позиция:** Нишевый, ориентирован на контент ГПМ
**Ссылка:** premier.one

### Что делают хорошо

**4K в браузере**
Один из немногих, кто поддерживает 4K-стриминг прямо в веб-браузере (не только в приложении).

**Сильный IP-контент**
Эксклюзивы ТНТ, ТВ-3, РЕН ТВ, СТС — для определённой аудитории это сильный магнит.

**25 оригинальных проектов в 2025 году**

### Что не работает

**Интерфейс — самый тёмный и загромождённый**
Описывается как "overly dark and cluttered" даже на фоне конкурентов с тёмными интерфейсами. Нет лёгкой темы.

**Ориентированность только на свой контент**
Каталог ощущается как промо-площадка контента ГПМ, а не полноценный стриминг.

### Что взять в NUX
- ✅ 4K поддержка в браузере — технический ориентир
- ❌ Не брать: ничего из UI/UX

## 6. START

**Позиция:** Растущий нишевый игрок, 26 оригинальных проектов в 2025
**Ссылка:** start.ru

### Что делают хорошо

**Чистый, современный интерфейс**
Start описывают как один из наиболее современных интерфейсов среди российских конкурентов. Минималистичный подход, без перегруженности.

**До 5 профилей по одной подписке**
Включая детский профиль (отсекает взрослый контент автоматически).

**Адаптивный стриминг**
Адаптивная битрейтовая передача данных — плавное воспроизведение даже на слабом соединении.

**Журнал / editorial раздел**
Есть редакционный блог "Журнал START" со статьями о кино — попытка editorial-подхода.

**Ставка на оригинальные проекты**
Yellow, Black and White — одна из сильнейших продакшн-студий России. Контент START делает команда, знающая своё дело.

### Что не работает

**Маленькая база фанатов**
START менее известен среди широкой аудитории, чем Кинопоиск или ivi.

**Поиск слабее лидеров**

### Что взять в NUX
- ✅ Концепция журнала / editorial раздела внутри платформы
- ✅ Минималистичный подход к интерфейсу
- ✅ Акцент на оригинальный контент как identity

## 7. ОБЩИЕ ВЫВОДЫ ПО РЫНКУ

### Что НЕ делает ни одна российская платформа (возможности для NUX)

| Пробел | Что есть сейчас | Что может сделать NUX |
|--------|----------------|----------------------|
| Editorial curation | Только алгоритмы или просто подборки | Человеческое редактирование как фича |
| Объяснение рекомендаций | "Вам может понравиться" без причины | "Потому что вы смотрели X режиссёра Y" |
| Лёгкая тема | Все только тёмные | Light/dark переключение (дизайн NUX поддерживает оба) |
| Нишевый арт-хаус | Никто не позиционирует себя как MUBI | Прямая ниша для NUX |
| Прозрачность без рекламы | Все смешивают платный/бесплатный контент | NUX: только одна подписка, весь контент одинаково доступен |
| Медиабейдж (тип контента) | Нет разделения Film/Doc/Game/Course | NUX: badge-система — дифференциатор |
| Социальные листы / watchparty | Только Кинопоиск (поверхностно) | Shared lists, "watching with" |

### Что хорошо делают все (стандарт, без которого нельзя)

- Тёмный интерфейс по умолчанию
- Офлайн-скачивание
- Resume (продолжить с места остановки)
- Детский профиль с PIN
- Поддержка Smart TV + мобайл

### Ключевые метрики из отзывов (что важно RU-пользователям)

1. **Качество видео/звука** — 54% (важнее всего)
2. **Цена подписки** — 50%
3. **Минимум рекламы** — 48%
4. **Удобный интерфейс** — важен, но не в топ-3 (берут за контент)
5. **Быстрый поиск** — боль у всех кроме ivi

**Вывод:** Для RU-аудитории контент > интерфейс. Но хороший интерфейс при хорошем контенте — сильное конкурентное преимущество, потому что ни у кого нет обоих одновременно.

## 8. КОНКРЕТНЫЕ ПАТТЕРНЫ ДЛЯ ЗАИМСТВОВАНИЯ

### От Кинопоиска
```
1. Inline-карточка фильма в тексте редакционной статьи
   └── [Постер] [Название] [Год] [Рейтинг] [Кнопка Watch]

2. Цветовой индикатор на карточке
   └── Зелёная точка = в подписке
   └── Монета = купить
   └── Серая = недоступно

3. Страница персоны (режиссёр / актёр)
   └── Все фильмы на платформе → linked cards
   └── Биография → внешняя ссылка
```

### От ivi
```
4. JSON дизайн-токены с абстрактными именами
   └── Цвета = города (criterion-amber, mubi-slate...)
   └── Один источник правды для всех платформ

5. Чат-онбординг
   └── Регистрация как диалог, а не форма
   └── Меньше визуальной сложности, больше вовлечённости
```

### От Okko
```
6. "2 клика" — принцип максимальной глубины навигации
   └── Любой контент: главная → раздел → тайтл (не глубже)

7. Воздушный интерфейс
   └── Большие отступы, меньше chrome
   └── Контент в фокусе, не UI
```

### Уникально для NUX (нет ни у кого)
```
8. "NUX Take" — редакционная заметка к каждому тайтлу
   └── 100–150 слов от команды
   └── Не synopsis, а точка зрения

9. Media-type badge система
   └── Film / Documentary / Game / Course
   └── Цветовой код на карточке

10. Двойная тема (Light/Dark)
    └── Единственная российская платформа с полноценным light mode
```
